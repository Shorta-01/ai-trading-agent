# Decision Log

This file is the **index** of all decisions made during the audit and
execution workflow. Each row references a detail file at
`docs/decisions/NNNN-<slug>.md` where the full context, options, and
consequences live.

Conventions:
- Append-only. Never edit or delete past entries.
- A new decision that replaces an old one is added as a new row; the
  superseding link is recorded in both rows' `Supersedes` and
  `Superseded by` columns.
- `Tasks` lists the `T-NNN` IDs of tasks generated by this decision,
  comma-separated.
