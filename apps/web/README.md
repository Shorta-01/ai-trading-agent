# Web (Next.js skeleton)

Eenvoudige Nederlandstalige frontend voor de technische startversie.

- Geen echte tradingdata
- Geen koop/verkoopadvies
- Geen brokerkoppeling
- Alleen papergeldmodus

## Lokaal starten

```bash
cd apps/web
npm install
npm run dev
```

## Controles

```bash
cd apps/web
npm run lint
npm run build
```
